## 一、什麼是 Streamlit？

👉 Streamlit 是一個可以**用 Python 做網頁小程式**的工具

## 二、數字輸入（number_input）

### 📌 功能

讓使用者輸入數字

### 🧸 例子在做什麼？

- 輸入一個 **0～100 的數字**
- 電腦會顯示你輸入的是多少

### ✨ 重點

- 可以設定：
  - 最小值
  - 最大值
  - 一次加多少（step）

## 三、成績等級判斷（if 判斷）

### 📌 功能

電腦會幫你看分數，告訴你等級

### 🎓 規則

| 分數    | 等級 |
| ------- | ---- |
| 90 以上 | A    |
| 80～89  | B    |
| 70～79  | C    |
| 60～69  | D    |
| 59 以下 | F    |

👉 **if / elif / else**
就像老師說：「如果這樣，就做這個」

## 五、重複做事情（for 迴圈）

### 📌 功能

讓電腦**一直做同一件事**

### 🔁 例子

```python
for i in range(5):
    print("Hello")
```

👉 會印出 **5 次 Hello**

### 🔢 range 的意思

| 寫法          | 結果    |
| ------------- | ------- |
| range(5)      | 0～4    |
| range(2,6)    | 2～5    |
| range(2,10,2) | 2,4,6,8 |

## 八、清單 List（很重要！）

### 📦 什麼是 List？

👉 一個可以裝很多東西的盒子

### 🧺 例子

```python
[1, 2, 3]
["apple", "banana"]
[1, "apple", True]
```

---

### 🔍 取出清單裡的東西

```python
L = [1, 2, 3, [4,5]]
```

| 寫法    | 結果  |
| ------- | ----- |
| L[1]    | 2     |
| L[3]    | [4,5] |
| L[3][0] | 4     |

---

### ✂️ 清單切片（切一部分）

```python
L = [1, 2, 3, "a", "b", "c"]
```

| 寫法   | 結果      |
| ------ | --------- |
| L[::2] | [1,3,"b"] |
| L[1:4] | [2,3,"a"] |

---

### 📏 len（長度）

👉 看清單裡有幾個東西

```python
len([1,2,3])  # 3
```

---

### 🔁 走訪清單（看每一個）

```python
for item in L:
    print(item)
```

## 九、清單常用指令

### ➕ 加東西

```python
L.append(4)
```

### ➖ 移除東西

```python
L.remove(1)
L.pop()
```

---

### ⚠️ 小心！

```python
a = [1,2,3]
b = a
```

👉 改 b，a 也會變
因為它們是**同一個盒子**
